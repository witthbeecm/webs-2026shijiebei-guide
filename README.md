# webs-2026shijiebei-guide

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为一组静态 HTML 文件的集合与分发点。所有页面均为独立编写，内容结构清晰，便于浏览与引用。

## 目录说明

```
webs-2026shijiebei-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面所需的静态资源（如图片、样式、脚本）
├── index.html      # 仓库首页（页面索引）
└── README.md       # 本说明文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，命名简洁，无特殊规则。
- `assets/`：资源文件按需存放，与页面通过相对路径引用。
- `index.html`：提供一个简单的导航列表，方便快速访问所有页面。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无依赖关系。每个页面均可直接通过浏览器打开，无需后端服务支持。页面内容保持原始状态，不做频繁更新，以确保归档的稳定性与可追溯性。

若需新增或修改页面，请遵循以下原则：

- 保持页面独立，不引入外部不可控资源。
- 页面代码应简洁、语义化，无冗余依赖。
- 提交时附带简要说明，描述新增或变更内容。

## 维护说明

本仓库由维护者定期整理与更新。所有页面均经过基本验证，确保可正常显示。

- 欢迎提交 Issue 报告页面问题或提出改进建议。
- 欢迎提交 Pull Request 贡献新页面或修复现有内容。
- 请勿在页面中插入任何形式的广告、追踪代码或恶意内容。
